# Day 01 Assignment

Find time complexity in each of the following cases:

## Q.01

```
for (i=1;i<=n;i++)
  for(j=i;i<=n;j++)
    printf("Hi");
```
## Solution:

Time Complexity of the code is O(n<sup>2</sup>)

## Q.02

```
for (i=1;i<=n;i*=3)
  for(j=i;i<=n;j++)
    printf("Hello");
```
## Solution:

Time Complexity of the code is O(n<sup>2</sup>)
